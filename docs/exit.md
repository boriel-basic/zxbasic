#EXIT

##Syntax

```
EXIT DO
EXIT FOR
EXIT WHILE
```

##Description

Terminates the current loop structure immediately and carries on execution from after the corresponding **LOOP**, **NEXT** or **END WHILE** statement.

##Remarks
* This is NOT compatible with Sinclair Basic
